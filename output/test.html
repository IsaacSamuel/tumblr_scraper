Anonymous asked: Is it true that it's forbidden in Iceland to use nicknames as given names?

Not exactly. Icelandic given names have to be on a list of given names approved by a committee; you can submit a name for review to the committee if it’s not already on the list, but to be approved it needs to obey Icelandic spelling, grammar and pronunciation rules. Thus, some nicknames are approved for use as given names, others not. This system may be changed in the near future, though.

risingdawn asked: Have you been using the buddy system on Pokemon Go? What do you think of it so far?

I have! It’s a nice step forward, both in terms of having some sort of interaction/bonding with your Pokémon and a way to get more candy for a Pokémon you only have one of, without completely disincentivizing catching more Pokémon. It’s extremely basic right now, though - I hope it gets expanded in the future. It’d be lovely if we could actually see the Pokémon following us on the map (admittedly requires walking animations), or interact with them in some Pokémon-Amie-like capacity.
